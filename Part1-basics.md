## 1.2 Integers multiplication

### Задача

Умножить два числа.

> Input:  
> Два N-значных не отрицательных числа X и Y
>
> Output:  
> Результат умножения X и Y

### Решение

Школьный алгоритм умножения столбиком.

```text
    5678
  x 1234
--------
   22712
  17034
 11356
 5678
--------
 7006652
```

### Сложность

Сложность в зависимости от длинны числа N.

1. Умножаем X на первую цифру Y  
   2N
2. Повторяем для всех цифр из Y  
   N * 2N
3. Складываем все результаты  
   N * 2N
4. Итого  
   2N^2 + 2N^2
